## 📋 Overview
This repository contains comprehensive Power BI dashboard for HR data analysis, uncovering valuable insights from workforce metrics.


## ⭐ Key Features
- 📈 Attendence analysis
- 💰 Presence insights
- 🎯 Attendence Trends over Time
- 🏢 Sick Leaves and Work From Home Trends
